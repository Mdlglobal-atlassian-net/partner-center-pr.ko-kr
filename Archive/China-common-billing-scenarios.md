# <a name="common-billing-scenarios"></a>일반적인 청구 시나리오

**적용 대상**

-   21Vianet에서 운영하는 파트너 센터


이 항목에서는 새 구독을 추가하거나, 구독에서 실제 사용자 수를 조정하거나, 구독을 취소한 후에 청구서에 표시될 내용에 대해 설명합니다.
