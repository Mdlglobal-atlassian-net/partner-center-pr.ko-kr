# <a name="partner-center-for-microsoft-cloud-germany"></a>Microsoft 클라우드 독일 파트너 센터

**적용 대상**

-  Microsoft 클라우드 독일 파트너 센터

## <a name="microsoft-cloud-germany-meeting-german-data-protection-requirements"></a>Microsoft 클라우드 독일: 독일 데이터 보호 요구 사항 충족 

Microsoft 클라우드 독일은 엄격한 독일 데이터 보호 요구 사항을 충족하도록 설계되었으며 독일 법률에 따라 고객 데이터에 대한 물리적 및 논리적 액세스를 제어하는 독일 데이터 수탁자를 보유하고 있습니다. 모든 고객 데이터 및 고객 데이터에 대한 액세스를 처리 또는 제어하는 시스템은 독일 데이터 센터에 상주합니다. 데이터 센터 간 네트워크는 격리되어 있으며 독일 내에서만 사용됩니다.

Microsoft 클라우드 독일은 독일의 데이터 센터를 통해 별도의 [Microsoft Azure](https://go.microsoft.com/fwlink/?linkid=847992), Office 365 및 Microsoft Dynamics 인스턴스를 제공합니다. 따라서 유럽 연합(EU) 및 유럽 자유 무역 연합(EFTA)에 소속된 상용 고객은 적용되는 독일 법률 및 규정과 주요 국제 표준에 따라 고객 데이터를 저장하고 관리할 수 있습니다. Microsoft 직원은 독일 데이터 수탁자 또는 고객의 승인과 감독 없이는 고객 데이터에 액세스할 수 없습니다.

자세한 내용은 [Microsoft 클라우드 독일 데이터 시트](http://download.microsoft.com/download/6/1/3/613C9ECB-9167-4EF5-B131-3BAD8D8A126C/Microsoft_Cloud_Germany_Datasheet.pdf)를 참조하세요.

## <a name="partner-center-for-microsoft-cloud-germany-selling-german-cloud-solutions-in-eu-and-efta"></a>Microsoft 클라우드 독일 파트너 센터: EU 및 EFTA 지역에 독일 클라우드 솔루션 판매

Microsoft 클라우드 독일 프로그램의 CSP(클라우드 솔루션 공급자)로 등록하면 파트너 센터를 사용하여 EU 및 EFTA 국가/지역의 고객에게 독일 클라우드 솔루션 및 서비스를 판매할 수 있습니다. 

>**참고**<br>
이 문서가 게시된 시점에는 Microsoft 클라우드 독일 파트너 센터에 다음과 같은 파트너 센터 기능이 제공되지 않았습니다(문서가 게시된 후 일부 기능이 파트너 센터에 추가되었을 수 있으며 이 목록이 최신 상태가 아닐 수 있음).

- 파트너 찾기
- 비즈니스 프로필
- 추천
- 라이선스 분석
- 고객 정보
- 알림 센터
- 간접 재판매인/공급자 모델 및 설정
- 서비스 상태 확인
- ISV(Independent Software Vendor) 등록
- 고객 특별 자격

개인 정보의 수집 및 사용에 대한 독일 법률 및 규정과 주요 해외 표준을 따르기 위해, 다음과 같은 파트너 센터 업무는 특정 서비스 포털을 통해 관리됩니다. 

기능 | 다음의 작업 완료
:--- | :---
파트너 사용자 관리 | Office 365 관리자 센터 또는 Microsoft 클라우드 독일 Azure 관리자 포털
고객 사용자 관리 | Office 365 관리자 센터 또는 Microsoft 클라우드 독일 Azure 관리자 포털
사용자 암호 다시 설정 | Microsoft 클라우드 독일 Azure 관리자 포털
라이선스 관리 | Office 365 관리자 센터 또는 Microsoft 클라우드 독일 Azure Active Directory 라이선스 포털

사용자 및 라이선스 관리에 대한 자세한 내용은 [Microsoft 클라우드 독일 파트너 센터에서 사용자 및 라이선스 관리](user-management-in-partner-center-for-microsoft-cloud-germany.md)를 참조하세요.

Microsoft 클라우드 독일 CSP 등록에 대한 자세한 내용은 [Microsoft 클라우드 독일 CSP 등록](enroll-in-csp-for-microsoft-cloud-germany.md)을 참조하세요.

## <a name="frequently-asked-questions-about-the-csp-program-for-microsoft-cloud-germany"></a>Microsoft 클라우드 독일을 위한 CSP 프로그램에 대한 질문과 FAQ

이러한 FAQ를 이용하여 Microsoft 클라우드 독일을 위한 클라우드 솔루션 공급자(CSP) 프로그램에 대한 자세한 내용을 확인하세요. 추가 정보는 고객 담당 관리자에게 문의하시기 바랍니다. 

**Microsoft 클라우드 독일이란?** 

Microsoft 클라우드 독일은 업계를 선도하는 Microsoft 클라우드 서비스의 인스턴스와 독일 내 고유한 시스템 관리 아키텍처 및 전용 네트워크가 결합된 것입니다. 이 고유한 모델을 기반으로 독립된 독일 기업이 '데이터 수탁자'로서 고객 데이터에 대한 액세스를 제어하고, 고객이 요청하거나 독일 법령에서 요구한 경우가 아닌 한 제3자에게 고객 데이터를 공유하지 않습니다. 이것을 '데이터 주권'이라고 합니다. 

**Microsoft 클라우드 독일의 CSP 프로그램 자격 대상은 누구인가요?**

유럽 연합(EU), 유럽 자유 무역 연합(EFTA) 및 영국의 파트너는 Microsoft 클라우드 독일 CSP 프로그램 자격 대상으로 EU, EFTA, 영국에서 고객을 상대로 판매할 수 있습니다. 

**CSP Commercial이라는 기존 파트너 프로그램이 있습니다. Microsoft 클라우드 독일 CSP 프로그램은 어떻게 다른가요?**

Microsoft 클라우드 독일 CSP 프로그램이 CSP Commercial과 다른 점은 다음과 같습니다. 
-   자격: Microsoft 클라우드 독일의 CSP 프로그램은 EU, EFTA, 영국의 파트너에게만 제공됩니다. 
-   등록: Microsoft 클라우드 독일 CSP 프로그램 파트너는 CSP Commercial과는 상관없이 별도 등록이 필요합니다. 
-   라이선스: Microsoft 클라우드 독일 CSP 프로그램의 라이선스 문서는 데이터 수탁자의 역할을 반영하는 내용으로 업데이트됩니다. 라이선스 변경에 대한 자세한 정보는 아래를 참조하세요. 
-   파트너 센터 기능: Microsoft 클라우드 독일 CSP 프로그램에서는 데이터 주권 요구사항을 준수하기 위해 일부 파트너 센터 기능을 제공하지 않습니다. 파트너는 Microsoft Online 포털(MOP), Azure Active Directory(Azure AD) 또는 Graph API를 대신 사용할 수 있습니다. 

**Microsoft 클라우드 독일 CSP 프로그램 라이선스는 CSP Commercial 라이선스와 어떻게 다른가요?**

Microsoft 클라우드 독일 CSP 프로그램의 Microsoft 클라우드 재판매인 계약(MCRA) 및 Microsoft 클라우드 배포자 계약(MCDA)은 다음의 두 가지 측면에서 Commercial MCRA 및 Commercial MCDA와 차이가 있습니다. 
-   고객은 독일 클라우드의 온라인 서비스 약관 외에 독일 클라우드에 적용되는 Microsoft 클라우드 계약(MCA) 수신이 요구됩니다. 
-   여기에는 고객 데이터 요구 사항의 수정이 포함됩니다. 클라우드 독일의 MCA는 다음의 두 가지 측면에서 사용 MCA와 차이가 있습니다. 
-   데이터 거주지에 대한 Microsoft의 추가적인 데이터 보호 조치, 데이터 수탁자의 역할, 고객 데이터 액세스를 포함하는 Online Services 약관 보충 조항을 포함합니다. 
-   데이터 수탁자 계약을 포함합니다. 보충 조항의 첨부 1에 포함된 고객 데이터 수탁자 계약에서는 T-System에서 고객 데이터 제어 및 액세스에 관련된 조치를 고객에게 직접 제공하도록 규정합니다. 

**Microsoft 클라우드 독일 CSP 프로그램의 판매 자격은 어떻게 되나요?**

Microsoft 클라우드 독일 CSP 프로그램 판매에 요구되는 자격 조건은 CSP Commercial 자격 조건과 동일하며, 직접 파트너와 간접 재판매인 모두 가능합니다. 이미 Commercial 영역에서 CSP 프로그램의 간접 재판매인으로 승인된 경우, Microsoft 클라우드 독일의 CSP 프로그램을 판매하기 위해 다시 간접 재판매인으로 추천받을 필요는 없습니다. 

**기존 파트너입니다. Microsoft 클라우드 독일 CSP 프로그램을 판매하려면 무엇이 필요한가요?**

파트너는 Microsoft 클라우드 독일 CSP 프로그램에 등록해야 합니다. (Microsoft 클라우드 독일 등록과 CSP Commercial 등록은 별도입니다). 등록 후에는 Microsoft 클라우드 독일 내에서 고객을 작성하고 제공되는 상품을 거래할 수 있습니다. API를 활용하는 경우 자세한 정보는 아래의 API를 참조하세요. 

**아직 CSP 파트너가 아닙니다. CSP 파트너가 되기 위해 자세한 정보는 어디에서 찾아볼 수 있나요?**

MPN 포털을 방문하면 파트너에게 제공되는 등록 모델에 대하여 자세히 알아볼 수 있습니다. 

**Microsoft 클라우드 독일 CSP 프로그램의 고객 제시 가격과 CSP Commercial의 고객 제시 가격은 동일한가요?**

아니요. 요금에는 Microsoft 클라우드 독일의 고유한 가치를 제공하기 위한 추가 비용이 반영됩니다. 

**Microsoft 클라우드 독일 CSP 프로그램의 파트너 할인 모델과 CSP Commercial의 파트너 할인 모델은 동일한가요?**

예. Microsoft 클라우드 독일 CSP 프로그램의 파트너 할인 모델은 CSP Commercial의 파트너 할인 모델과 동일하게 제공됩니다. 

**CSP Commercial과 비교했을 때 Microsoft 클라우드 독일 CSP 프로그램에서 제공되는 파트너 인센티브가 다른 점이 있나요?**

아니요. Microsoft 클라우드 독일을 위한 CSP 프로그램의 파트너 인센티브는 CSP Commercial의 파트너 인센티브와 동일합니다. 파트너 인센티브에 대한 자세한 정보는 MPN 포털에서 최신 정보를 참조하세요. 

**CSP Commercial과 비교했을 때 Microsoft 클라우드 독일 CSP 프로그램의 청구 방식에 다른 점이 있나요?**
아니요. Microsoft 클라우드 독일의 CSP 프로그램의 청구 조건과 요구사항은 CSP Commercial과 동일합니다. 청구에 대한 자세한 내용은 MPN 포털의 리소스 섹션을 참조하세요. 

**Microsoft 클라우드 독일의 CSP 프로그램 지원 요구사항과 CSP Commercial의 지원 요구사항은 동일한가요?**

예. 지원에 대하여 자세히 알아보려면 MPN 포털의 리소스 섹션을 참조하세요. 

**판매가 가능한 대상 고객은 어떻게 되나요?**

CSP MCD 파트너로 승인된 후에는 EU, EFTA, 영국의 모든 상업 고객을 대상으로 판매할 수 있습니다. 

**Microsoft 클라우드 독일 CSP 프로그램에서 제공되는 제품/서비스는 무엇인가요?**

출시 시점에서는 Microsoft Office 365 Germany 및 Microsoft Azure Germany가 제공됩니다. Microsoft Dynamics 365 Germany(CRMOL)는 2017 회계년도 2분기로 계획된 상태입니다. 이후 준비되는 대로 다른 서비스를 추가할 예정입니다. 

**Microsoft 클라우드 독일 CSP 프로그램에서 모든 Azure 서비스가 제공되나요?**

현재 IaaS 및 기본 플랫폼과 데이터 상품이 제공되고 있습니다. 로드맵 업데이트 현재 진행 중이며, 파트너 센터를 통해 CSP Commercial 문서와 같은 문서를 제공할 예정입니다(인증 필요). 

**Microsoft 클라우드 독일 CSP 프로그램을 사용해 개발하려면 무엇이 필요한가요?**

Microsoft 클라우드 독일을 사용한 개발에 대하여 자세히 알아보려면 독일 클라우드 파트너 센터 사용 개발을 클릭하세요. 

**Microsoft 클라우드 독일 CSP 프로그램과 CSP Commercial의 API 간에 차이점이 있나요?** 

Microsoft 클라우드 독일 CSP 프로그램의 API 및 통합 요구사항은 CSP Commercial API 및 통합 요구사항과 동일합니다. Microsoft 클라우드 독일 끝점에 대한 자세한 정보는 파트너 센터 REST URL에서 확인할 수 있습니다. 

참고로 Microsoft 클라우드 독일 CSP 프로그램에서는 CREST API를 사용할 수 없습니다. 현재 CSP Commercial에서 해당 API의 사용 중단이 진행되고 있기 때문에 Microsoft 클라우드 독일에서는 제공되지 않습니다. 다음은 Microsoft 클라우드 독일을 위한 CSP 프로그램에서 제공되지 않는 API입니다. 이는 데이터 주권 요구사항을 준수하기 위한 것입니다.
