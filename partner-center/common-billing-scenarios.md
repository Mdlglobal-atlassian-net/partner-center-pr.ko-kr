# <a name="common-billing-scenarios"></a>일반적인 청구 시나리오

**적용 대상**

-  파트너 센터
-  Microsoft 클라우드 독일 파트너 센터

이 항목에서는 새 구독을 추가하거나, 구독에서 실제 사용자 수를 조정하거나, 구독을 취소한 후에 청구서에 표시될 내용에 대해 설명합니다. 사용량 기준 및 라이선스 기준 구독에 미치는 영향은 각기 다릅니다.

## <a href="" id="usagebased"></a>사용량 기준 청구


이전 청구 기간에 사용된 서비스에 대해서만 요금이 청구됩니다. 청구 기간 동안 활성화되고 사용된 Azure 서비스 또는 응용 프로그램이 송장에 표시됩니다.

-   첫 번째 청구서에는 구독 시작 날짜부터 첫 번째 청구 날짜까지의 초기 기간이 포함됩니다.
-   데이터 통신 서비스 요금은 송장 주기 내에서 변경할 수 있습니다.
    -   가격 상승: 30일 알림이 제공됩니다.
    -   가격 하락: 변동 기간(일)을 반영합니다.
    -   기존 구독은 청구서 주기 시작 시 유효한 요금을 사용합니다.
    -   새 구독(청구 주기 내에 만들어진 경우)은 만든 날짜에 유효한 요금을 사용합니다.
-   첫 번째 청구 마감일 이전이나 청구 주기 중간에 구독을 취소하는 경우 조정 파일에 구독이 활성화된 기간 동안의 사용 요금이 표시됩니다.
