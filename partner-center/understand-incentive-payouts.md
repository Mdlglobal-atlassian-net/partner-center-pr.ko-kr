# <a name="view-your-incentive-payments-and-programs"></a>동기 지불 및 프로그램 보기

**적용 대상**

-  파트너 센터

**적절한 역할**

-  동기 관리자
-  인센티브 사용자 
-  동기 읽기 전용 뷰어
-  글로벌 관리자
-  MPN 파트너 관리자

이러한 페이지를 사용 하 여 과거, 보류 중, 예정 된 지불액과 소득, 각 위치에서 동기 프로그램의 상태 및 전체 트랜잭션 기록에 대 한 세부 정보를 볼 수 있습니다. 또한 모든 트랜잭션에 대 한 자세한 정보를 다운로드 하거나 내보낼 수 있습니다. 모든 성과급 페이지에는 미국 달러 단위로 금액이 표시 됩니다.

>[!NOTE]
>연결 된 MPN Id 및 프로그램에 대 한 데이터만 볼 수 있습니다. 추가 데이터에 액세스 하려는 경우에는 성과급 관리자에 게 문의 하세요. 

## <a name="access-the-incentives-pages"></a>성과급 페이지 액세스

파트너 센터 [대시보드에서](https://partner.microsoft.com/dashboard)몇 가지 방법으로 성과급 정보에 액세스할 수 있습니다.

- 왼쪽 메뉴에서 **성과급** 을 선택 합니다. **개요**를 비롯 한 몇 가지 옵션이 **성과급**아래에 나타납니다. 이러한 옵션을 탐색 하 여 동기 프로그램, 동기 상환, 수입 및 클레임의 상태를 알아보세요.
- 오른쪽 위 모서리에서 **Money** (지급) 아이콘을 선택 합니다. 그런 다음 **트랜잭션 기록**, **지불**, **데이터 내보내기** 또는 **Office 판매 보고서**와 같은 기타 옵션을 선택할 수 있습니다.

다음 페이지는 여기에 설명 되어 있습니다.

- 내 성과급 개요 페이지
- 지불 페이지
- 트랜잭션 기록 페이지
- 데이터 페이지 및 관련 된 기타 내보내기 페이지 내보내기

>[!NOTE]
>파트너 센터의 성과급 및 동기 기능에 대 한 자세한 내용은 [파트너 투자 및 성과급](https://partner.microsoft.com/membership/partner-incentives) (로그인 필요)을 참조 하세요.

## <a name="my-incentives-overview-page"></a>내 성과급 개요 페이지

**내 성과급 개요** 는 각 위치에서 각 프로그램의 상태 뿐만 아니라 동기 프로그램의 전반적인 상태를 보여 줍니다. 이 정보에는 동기 프로그램에 등록 한 이후의 총 동기 수입 및 지불이 포함 됩니다.

또한이 페이지에는 특정 위치에서 프로그램에 등록 하기 위해 수행 해야 하는 추가 작업 뿐만 아니라 위치 또는 프로그램에의 한 소득 및 지불액이 포함 되어 있습니다.

**내 성과급 개요**에 액세스 하려면:

1. 파트너 센터 [대시보드에](https://partner.microsoft.com/dashboard)로그인 합니다.
2. 메뉴에서 **성과급**, **개요** 를 차례로 선택 합니다.
3. 페이지의 맨 위에서 소득 및 지불 요약을 확인 하 고 아래 표에 상세 정보를 표시 합니다. 또한 해당 테이블을 정렬 하 고 그룹화 하 고 확장할 수 있습니다.

   - 열을 기준으로 정렬 하려면 열 이름을 선택 합니다.
   - 프로그램 별로 그룹화 하려면 표 위의 **프로그램** 탭을 선택 합니다.
   - 위치별 그룹화 하려면 테이블 위의 **위치** 탭을 선택 합니다.
   - 특정 그룹 내의 등록에 대 한 자세한 정보를 보려면 지정 된 행의 끝에서 갈매기형 수장 기호를 선택 합니다. 이렇게 하면 뷰가 확장 됩니다.

4. 프로그램에 등록 하는 데 추가 작업이 필요한 경우이 정보가 **상태** 열에 표시 됩니다. 이 경우 다음 단계를 수행 해야 하는 다음 단계에 대 한 자세한 내용을 보려면 갈매기형 펼침 기호를 선택 합니다.

>[!NOTE]
>**상태** 열에 표시 되는 다양 한 등록 상태에 대 한 자세한 내용을 보려면 다음 표를 참조 하세요.


| **상태**         | **표시 되는 경우** |
|:------------------------------------|:------------------|
| 필요한 작업  | 파트너가 동기 프로그램에 등록할 수 있는 초대를 수락 했지만 은행이 나 세금 정보를 업데이트 해야 할 수 있습니다. 파트너 센터에서 은행 또는 세금 정보를 업데이트 하려면 다음 단계 또는 링크에 대 한 **작업 필수** 열을 참조 하세요. |
| 단종  | 특정 동기 프로그램은 더 이상 성과급 시스템에서 제공 되지 않습니다. |
| 등록  | 모든 세금 및 은행 정보에 대 한 유효성이 검사 되었습니다. 파트너는 추가 등록 작업을 수행할 필요가 없습니다. |
| Inactive  | 이 시점에는 동기 프로그램이 등록에 열려 있지 않거나 파트너가 등록 또는 다시 등록에 대 한 현재 자격을 충족 하지 않을 수 있습니다.  |
| 초대됨  | 파트너에 게 새 동기 프로그램 등록 초대를 보냈지만 파트너가 아직 등록 프로세스를 시작 하지 않았습니다. 인접, **작업 필요** 열은 다음 단계와 모든 관련 링크를 표시 합니다.  |
| 등록 유효성 검사  | 파트너가 신규 또는 기존 등록에 대 한 은행 및 세금 정보를 이미 완료 했거나 업데이트 했으며 Microsoft에서이 정보를 확인할 때까지 기다리고 있습니다. 유효성 검사 프로세스 중에 최대 48 시간 동안 **등록 유효성 검사가** 표시 될 수 있습니다.  |

## <a name="payments-page"></a>지불 페이지
이 페이지의 합계는 사용 하는 모든 MPN Id를 나타냅니다. 참가자 ID, 프로그램, 지불 ID 및 획득 유형을 기준으로 필터링 할 수 있습니다. 금액은 미국 달러 단위로 제공 됩니다. 유료 값은 지불 금액에도 표시 됩니다. 

|**영역**   |**설명**    |
|------------------|:-------------------------------------|
|올해 총 지불 금액        |모든 MPN Id에 대해 미국 달러 단위로이 연도에 대해 지급 된 총 합계입니다.                                      |
|다음 예상 지불      |곧 귀하에 게 제공 되는 한 번의 지불 (다른 사용자가 곧 제공 되는 경우에도)은 미국 달러입니다.                                     |
|마지막 지불           |가장 최근의 지불의 양 (미국 달러), 프로그램 이름 및 MPN ID입니다.                                      |
|원본 별 지불       |지난 12 개월 동안 프로그램으로 표시 된 지불 금액 (미국 달러)입니다.                                      |
|지급                       |**유료** 또는 **보류 중** 을 선택 하 고 원하는 대로 정렬 합니다. 특정 결제에 대 한 추가 정보는 **보기**를 선택 합니다. 결제 송금 문의 사본을 다운로드 하려면 **다운로드**를 선택 합니다. 트랜잭션 기록 데이터를 표시 하는 데 최대 24 시간이 걸릴 수 있으므로 관련 소득을 즉시 볼 수 없습니다.  |

이 페이지에서 데이터를 내보내려면 **내보내기** 를 선택한 다음 데이터 내보내기 페이지의 지시를 따릅니다. 

## <a name="transaction-history-page"></a>트랜잭션 기록 페이지
이 페이지에는 각각에 대 한 날짜, 유형 및 획득을 포함 하 여 개별 소득이 모두 표시 됩니다. 보려는 기간을 선택할 수 있으며 등록 ID, 프로그램, 지불 ID, 획득 유형, 레버 및 상태별로 필터링 할 수도 있습니다. 현재 회계 연도 (7 월 1 일-6 월 30 일)와 이전 두 회계 연도에 대해 데이터를 사용할 수 있습니다. 

획득에 대 한 자세한 내용을 보려면 페이지의 오른쪽에 있는 아래쪽 화살표를 선택 합니다. 그러면 레버, 수익 금액, 제품 및 고객이 표시 됩니다. 어떤 이유로 든이 데이터를 사용할 수 없지만 액세스 권한이 필요한 경우 지원 담당자에 게 문의 하세요. 트랜잭션이 아닌 조정 결과가 발생 하는 경우에는 Product 및 Customer 필드가 표시 되지 않습니다. 

이 페이지에서 트랜잭션 데이터를 내보내려면 **내보내기** 를 선택한 다음 데이터 내보내기 페이지의 지시를 따릅니다. 트랜잭션 기록 페이지에서 내보낸 파일에는 트랜잭션 통화, 거래 통화 및 미국 달러의 수익, 유료 통화의 유료 값 등의 데이터가 표시 됩니다.

## <a name="export-data-page"></a>데이터 내보내기 페이지
이 페이지의 지침에 따라 원하는 데이터를 내보냅니다. 

**참고:**
-   연결 된 MPN Id 및 프로그램에 대 한 데이터만 볼 수 있습니다. 추가 데이터에 액세스 하려는 경우에는 성과급 관리자에 게 문의 하세요. 
-   데이터 내보내기 페이지는 자체적으로 새로 고쳐지지 않습니다. 최신 데이터를 보려면 페이지를 수동으로 새로 고쳐야 할 수도 있습니다. 
-   필터를 사용 하면 데이터를 **사용할 수 없음** 오류가 발생할 수 있습니다. 이는 세 달에 선택한 기본 기간을 떠난 다음 해당 기간 외의 획득에서 지불 ID를 선택 했음을 의미 합니다. 기간을 확장 하 고 다시 시도 하세요.

|programName   |동기/스토어 프로그램 이름      |
|이율   |해당 프로그램/participantID의 지불 금액에서 획득 한 금액      |
|Ened Usd   |프로그램/참가자 ID의 획득 금액 (USD)      |
|withheldTax   |Program/participantID에 대 한 종 량 제 보안상 이유로의 세금      |
|salesTax   |Program/participantID에 대 한 종 량의 총 판매 세금      |
|serviceFeeTax  |프로그램/participantID에 대 한 유료 통화의 총 serviceFeeTax 크기 |

|currencyCode   |통화 코드 요금 지불      |
|paymentMethod   |파트너를 지불 하는 데 사용 되는 방법 (전자 은행 전송, 크레딧 메모)      |
|paymentID   |지불의 고유 식별자입니다. 이 번호는 일반적으로 은행 명세서에 표시 됩니다.      |
|paymentStatus   |지급 상태      |
|paymentStatusDescription   |지불 상태 설명      |
