# <a name="billing-overview"></a>청구 개요

**적용 대상**

-  파트너 센터
-  CSP 프로그램의 파트너

제품, 솔루션 및 서비스 고객을 대신 하 여 구입에 따라 다음 방법 중 하나 이상의 이러한 구매에 대 한 청구 됩니다.
-   [라이선스 기준 청구](#licensebasedbilling)

    라이선스를 필요로 하는 온라인 서비스 또는 제품을 구입 하면 (라이선스 사용)에 없는 구입한 각 라이선스에 대 한 요금이 청구 됩니다. 한 달에 한 번 또는 1 년에 두 번 청구 여부를 선택할 수 있습니다. 비즈니스 요구 변화 하는 경우에 다른 간에 전환할 수 있으며 다시 수 있습니다. 
    
    월별 및 연간 청구에 대 한 자세한 내용은 청구 [FAQ](https://docs.microsoft.com/en-us/partner-center/faq-about-new-billing-features)를 참조 하세요.

-   [사용량 기준 청구](#usagebasedbilling)

    Azure 구독 같은 온라인 서비스를 구입 하면 월별 사용률에 대 한 요금이 청구 됩니다. 만 월별 청구는 사용량 기준 제품에 사용할 수 있습니다. Azure와 같은 사용량 기준 서비스를 기준으로 데이터 통신 연결된 속도 따라 요금이 청구 됩니다.

-   [일회성 청구](#onetimebilling)

    Azure 예약 이나 다른 소프트웨어 구독을 구입 하면 미리 설정된 된 기간에 대해 미리 지불 합니다. 미리 지불할 때문에 하나의 총액에 요금이 청구 됩니다. 
    
월별 청구를 선택한 경우 또는 매월 청구 되는 사용량 기준 제품을 구입 하는 경우 월별 청구 날짜는 파트너 센터에서 CSP 계좌를 만들 때 선택한 달의 일입니다. 성공적으로 만든 후 CSP 계정, Microsoft는 청구 날짜가 포함 된 확인 메일을 보냅니다. 만들어진 경우이 날짜를 변경할 수 없습니다. 

## <a name="pricing-and-invoicing"></a>가격 책정 및 송장 작성
가격표는 매달 업데이트되므로 한 달 전에 확인할 수 있습니다. 라이선스 기준 가격은 구독 기간(일반적으로 구매 날짜부터 12개월) 동안 보장됩니다. 사용량 기준 가격은 월 단위로 변경될 수 있습니다. 

하지만 제품, 서비스 및 소프트웨어 구독에 대 한 가격은 구독 기간을 통해 보장 갱신 하면 가격 변경 될 수 있습니다.

크레딧 또는 조정은 적용된 후 다음 청구 송장에 조정 및 크레딧이 후불로 표시됩니다.

파트너 센터의 청구 페이지에서 송장 및 조정 파일을 보고 다운로드할 수 있습니다. 월별 송장은 파트너 센터에서 선택한 청구 날짜 4일 이내에 제공됩니다.

## <a name="payment-terms"></a>지급 조건

지급 조건 순 60 일입니다. 하 여 송장 요금을 지불 해야 송장 기한 (청구 날짜 후 60 일) 또는 계좌 연체 됩니다, CSP에 등록에 영향을 줄 수입니다. 이전 지불할 때 일시 중단 된 계정의 모든 기능을 다시 사용할 수 예정 금액입니다.

### <a name="tax"></a>세금

사용자는 성립 기준으로 세부 정보를 (고객이 아니라 귀하) 청구 관계는 Microsoft와 귀하 사이입니다. 계정 설정 프로세스 중 이나 나중 지원 요청을 제출 하 여 세금 ID를 제출할 수 있습니다. 다음 청구 주기에 변경 내용이 반영된 것을 확인할 수 있습니다.

-   원천 징수 세 및 판매세 면 세 증명서에 대 한 지원 요청을 통해 세금 서류를 제출 해야 합니다. 다음 청구 주기에 변경 내용 및 적절한 환불을 확인할 수 있습니다.

-   VAT(부가가치세) 면세의 경우 서비스 요청을 통해 VAT ID(Microsoft에서 확인함)를 제출해야 합니다. 다음 청구 주기에 변경 내용 및 적절한 환불을 확인할 수 있습니다.

현지 세무서 또는 세금 고 문에 게 세금 정보 추가 찾을 수 있습니다.

## <a href="" id="licensebasedbilling"></a>라이선스 기준 청구

고객을 대신 하 여 라이선스 기반 제품을 구입 하면 월별 또는 연간 청구를 선택할 수 있습니다. 나중에에 청구 주기를 변경 하려는 경우 다음 절차를 사용 합니다. 

월별 청구에서 연간 청구로 전환 매월 청구 되는 다양 한 구독 있고 일반적인 청구 날짜에 맞게 하려는 경우 유용 합니다. 월별 청구를 연간 청구에서 전환 하는 것은 개별 고객의 청구 날짜 조정에 유용 합니다. 

청구 주기를 변경 하면, 용어 연간 청구 주기를 변경한 날짜와 갱신 날짜 설정 된 반영 하도록 업데이트 됩니다. 

비즈니스가 변경 될 때마다 청구 주기를 변경할 수 있습니다. 

### <a name="billing-rules-for-annual-billing"></a>연간 청구에 대 한 청구 규칙

-   구독은 연간 단위로 이루어지며 자동 갱신됩니다.

-   청구 12 월별 지급 또는 연간 구독 당 하나의 연간 결제입니다.

-   라이선스 기준 서비스의 경우 이전 청구 기간 종료 시 라이선스의 수에 따라 다음 청구 기간에 대한 요금이 사전 청구됩니다.

-   라이선스 수 변경 시 후불로 청구/입금됩니다(라이선스-일수를 기준으로 비례하는 계산). 비례하는 계산에서는 [ROUND((ROUND(단가 * 수량 / 비례하는 달의 일수, 2) * 비례하는 일수) / 수량, 2) * 수량] 수식을 사용합니다.

-   결제 (하지 라이선스 프로 비전)을 판매 된 라이선스에 대 한 청구 됩니다.

### <a name="to-change-the-billing-frequency-of-an-online-service"></a>온라인 서비스의 청구 주기를 변경 하려면

1.  파트너 센터 메뉴에서 **고객** 을 선택 하 고 변경 하려는 구독을 사용 하 여 고객을 선택 합니다. 

2.  고객의 구독 페이지에서 변경 하려는 구독을 선택 합니다. 

3.  세부 정보 페이지의 **청구 주기**를 **월별** 또는 **연간**를 선택 합니다. 변경할 구독 목록을 뿐만 아니라 청구 주기를 변경 하는 방법에 대 한 중요 한 정보는 확인 페이지가 표시 됩니다. 

4.  **확인** 을 변경 하거나 실행 취소 **취소** 를 선택 합니다. 

### <a name="adjustmentscreditscancellations"></a>조정/크레딧/취소

Microsoft은 라이선스 기준 서비스의 취소를 위한 조기 종료 수수료가 부과 하지 않습니다. 

라이선스 기준 서비스의 취소 크레딧은 중도 취소에 사용되지 않은 일수(와 위의 수식에 따른 라이선스 감소)에 비례합니다.

## <a href="" id="usagebasedbilling"></a>사용량 기준 청구

일부 Microsoft 제품 및 서비스 사용 되는 서비스에 대해서만 청구는 "량" 청구 모델을 사용 합니다. 예를 들어 Microsoft Azure이이 모델을 사용합니다. 

### <a name="billing-rules"></a>청구 규칙
-   구독 달을 되며 새로운 데이터 통신된 요금으로 자동으로 갱신 됩니다. 매달 이전 달의 사용에 대 한 요금이 청구 됩니다.

-   데이터 통신 서비스 요금은 송장 주기 내에서 변경할 수 있습니다. 

    -   가격 인상: 30일 동안 공지가 제공됩니다.

    -   가격 하락: 변동 기간(일)을 반영합니다.

    -   기존 구독은 청구서 주기 시작 시 유효한 요금을 사용합니다.

    -   동일한 청구 주기 내에서 만들 때 새 구독을 만든 날짜에 유효한 속도 사용 합니다. 

### <a name="adjustmentscreditscancellations"></a>조정/크레딧/취소

다음 월별 청구 송장에 조정이 있는 결제가 표시됩니다.

Microsoft은 사용량 기준 서비스의 취소를 위한 조기 종료 수수료가 부과 하지 않습니다. 

다음 월별 청구 송장에 SLA 크레딧을 비롯한 모든 유형의 크레딧이 표시됩니다.

## <a href="" id="onetimebilling"></a>일회성 청구

1 년 또는 3 용어에 대 한 소프트웨어 구독 및 Azure 예약을 사전에 구입할 수 있습니다. 미리 구입 하면 한 총액에서 전체 비용에 대 한 요금이 청구 됩니다. 이 유형의 청구 일회성 청구를 이라고 합니다.

>[!IMPORTANT]
>Azure 예약 및/또는 통화를 사용 하 여 위치에 있는 고객에 대 한 소프트웨어 구독을 구매 하면 청구 통화 기본 위치가 아니라 고객의 위치를 기반으로 합니다. 여러 위치에서 고객을 보유 하 고 별도 송장 받게 됩니다 청구 필요한 각 통화에 대해 조정 파일을 적절 한 통화로 고객에에서 게를 할 수 있도록 합니다. 

### <a name="manage-your-one-time-billing"></a>일회성 청구 관리

**현재 청구 상태, 송장 및 조정 파일 보기**

1.  파트너 센터에서 **청구** 와를 차례로 선택 **한 번** 에 청구 상태를 확인 합니다. 

2.  자세한 정보를 보려면 송장 또는 조정 파일을 선택합니다. 

**고객의 주문 내역 보기**

1.  파트너 센터 메뉴에서 **고객** 을 선택 합니다.

2.  **고객** 페이지에서 주문 내역을 확인하려는 고객을 찾고 아래쪽 화살표를 선택하여 고객의 기록을 펼쳐봅니다. 

3.  **주문 보기**를 선택하여 주문 내역을 표시합니다.

**신용 메모를 다운로드 합니다.**

신용 카드나 재 청구를 요청 해야 할 경우 신용 메모 원래 송장 취소를 제공 해 드리겠습니다. 다음과 같은 이유로 신용/재 청구를 요청할 수 있습니다.

-   주소 또는 구매 주문 수정

-   송장 생성 하 고 세금 환불 적용 합니다. 신용/재 청구 가져오려면 원래 송장에 다시 값을 가져오는 세금 환불을 요청할 수 있습니다. 그 다음으로 원래 송장 환불에서 풀의 신용/있으면 요청 수에, 환불에 대 한 마찬가지입니다.
